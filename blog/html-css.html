What does it mean to display inline vs inline-block?

When we are writing content for a website, we wrap text in various tags to label it and give it different significance. Each element appears within a rectangular box and has its own display properties by default. (These rectangular boxes can be made visible by changing the value of the border property of each element). Understanding the way that different elements behave when a web page is loaded is very useful when it comes to designing the layout of your website.

Inline elements

Inline elements like <a>, <span>, <img> and <button> have a number of default properties to be aware of when considering using them in your website:

  • They take up as much space as needed
  • They can be placed side by side, in the natural flow of the web page
  • You can not set the width and height of these elements

Block-level elements

Block-level elements like <div>, <p>, <h1>, <ul> and <footer> have default properties of their own and also have their own limitations. These properties include:

  • They will start on a new line
  • By default, they will take up the full width of the page
  • You can set the width and height of these elements

Keep in mind that these are the default properties of these elements. If we want to manipulate these elements, we are able to assign different values to each element’s display property which will change the way they behave.

Inline-block elements

Alternatively, we can assign the inline-block value to an element’s display property which means the element will take on the behaviour of a block-level element while still appearing the natural flow of the web page. Inline-block properties behave like so:

  • They take up as much as space as needed by default
  • They can also have their height and width set
  • They appear in the natural flow of the web page

This means that elements with a display property value of inline-block are more versatile in some situations where you need to manipulate the width and height of an element, while still appearing in the natural flow of the web page. The display property is very important when it comes to designing a web page. There are all sorts of ways you can manipulate the way each element appears. It is important to know the default display value of each element and how to set a new value and have the element behave the way you want it to.
